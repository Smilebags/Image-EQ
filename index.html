Image EQ

This is the input image. You can use this or use your own.

This is still a work in progress, my final plan is to have an EQ style graph with controllable points in order to push/pull certain frequencies in the image. This feature isn't available in any major software so I thought I would develop it myself.

This was made using standard JavaScript, including a modified version of Kevin Kwok's code for FFT and DCT for some of the basis for the behind the scenes magic. It could be made significantly faster by moving the heavy calculations to the GPU or alternatively to WASM, but I don't have the math skills to do this.
